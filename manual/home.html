This git repository is used for a live pattern recognition of coin sound signals.
The coins are thrown into a bowl and the signal is recorded by a microphone array.
The plan shows how to place the microphones and the bowl in 3 variations with 2 bowls. The coins are thrown through the slot above the bowl.

How to perform live pattern recognition Keyword Index Function Index
For prepairing a live experiment follow the steps below, if you have already trained models you can start at step 3.
  1. Execute the FEA.xtp script in the directory of the actual experiment with following parameters:
    • $1 ana
    • $2 [UASR_HOME]-data/coins/[EXPERIMENTNAME]/info/[EXPERIMENTNAME].mcfg
    Replace [UASR_HOME] with the path to your UASR-directory or the according system variable
    and [EXPERIMENTNAME] with the name of your experiment.
  2. Execute the HMM.trn script in the same directory with following parameters:
    • $1 trn
    • $2 [UASR_HOME]-data/coins/[EXPERIMENTNAME]/info/[EXPERIMENTNAME].mcfg
    The 2nd Parameter has to be equal as in the FEA.xtp script, also be sure to use absolute file paths.
  3. Execute the REC_PACKDATA.trn script in the same directory with following parameters:
    • $1 rec
    • $2 [UASR_HOME]-data/coins/[EXPERIMENTNAME]/info/REC_PACKDATA.cfg
    • $3 -v2
    The Path ($2) of the config-file is relative to the working directory and has to define the working model.
  4. Now run the life experiment with the packed data.
